## Forms of circle equations

### General or Implicit Form

```math
x^2 + y^2 + Ax + Bx + C = 0
```

### Standard Form

```math
(x - a)^2 + (y - b)^2 = r^2
```

### Parametric Form

```math
x = h + r\cos(\theta)
```
```math
y = h + r\sin(\theta)
```
